Permission Setup Wizard Android module

This folder contains a minimal Android app module that implements a "Permission Setup Wizard".

How to use
1. Open Android Studio and import this module as a new project or add it to your existing Android project.
2. Set your `applicationId` and configure signing if you want to run on device.
3. Add the following environment settings when deploying any server functions referenced in the app (if used): none required for local testing.

Notes
- This module is provided as a helper starting point. You should review all permissions and ensure compliance with Play Store policies before publishing.
